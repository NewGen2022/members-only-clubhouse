# Technologies used
- ExpressJS
- EJS (for frontend/templating)
- PostgreSQL
- CSS for styling
- adaptable.io to deploy everything

# Functionality of the app
- Sign Up
- Log In
- Log Out
- Become club member (Join club)
- Become admin
- Create Message
- View when and who added a message (as club member)
- Delete message (as admin)
